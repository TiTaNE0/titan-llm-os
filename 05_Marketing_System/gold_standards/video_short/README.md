# Gold standards — video_short

The Judge scores subjective criteria (voice-match, save/share) by **similarity to these gold standards**, not abstract ideals (§10.2). Gold standards are **per (channel × content_type)** — this set is `video_short` only; carousel/x_post each need their own set before they can be judged.

## Status: SEEDED (provisional) — 2026-06-02
Audience is set (US-inbound international-student wedge), so three niche-pattern exemplars are seeded: `ex01_fahrenheit_forecast`, `ex02_oven_recipe`, `ex03_fever_reading` (weather / kitchen / health — the three densest daily-friction pains). **They are PROVISIONAL** — structural pattern exemplars, not verified live TikTok links (could not browse TikTok). The Judge uses them to calibrate criteria 3 (voice) & 5 (save/share). **Replace with REAL high-performing links once the campaign ships and ANALYTICS surfaces proven winners (§8.5).**

## What goes here (when seeded)
- 3–8 real, high-performing `video_short` pieces in the actual niche/audience — each a short file: link, why it wins (hook, manifest-clarity, voice, CTA, save/share trigger), and which rubric criteria it exemplifies.
- Initially **niche/channel winners**; over time **replaced/augmented by the campaign's own proven pieces** (§8.5) — winners from ANALYTICS get promoted here.

## How winners get promoted (post-launch)
A piece that performs well in ANALYTICS → add it here as an objective gold standard → the Judge calibrates to *your* proven results, not borrowed ones.
