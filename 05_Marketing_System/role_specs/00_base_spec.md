# Base Pipeline Spec (all agents inherit)

Every agent loads this first. It is a **specification, not a personality** — decision criteria and contracts, never a character or voice.

## 1. No operator voice (§8.2)
Voice is derived per (channel × audience × trend) by the Scout — never the operator's personal style. The **audience is CANON (operator-set); the voice is discovered.** Content promotes a *product to an audience*, not the operator.

## 2. Quality bar
Clarity over cleverness. Specificity over hype. Usefulness over reach-bait.

## 3. Handoff discipline
- Read ONLY the vault files you're given; write ONLY your contracted output.
- Structure over prose at every seam (the schema is the tone-firewall — §8.6).
- All handoffs go through vault files, not model-to-model prose.

## 4. CANON is constant
Never silently override CANON. A conflict is **surfaced as BREAK** (halt + notify operator), not patched.

## 5. Cost awareness
Frontier tokens are spent only on directions that survive the Gatekeeper. Be concise; the trace logs your tokens.
