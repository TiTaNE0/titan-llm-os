# Article Module

**Status:** inactive
**Channel:** Long-form blog (Substack / personal site / X Articles standalone)

## Scope

Reserved slot for long-form prose pieces (≥1500 words) that aren't tied to a Twitter thread. Activate with `/enable_module article` when ready.

When activated, expected files:
- `README.md` (this file, expanded)
- `strategy.md` (audience, length targets, structure conventions)
- `templates/` (essay outline, hands-on review, post-mortem, etc.)
- `failure_log.md`

Voice file at `05_Content/personalization/voice_evgeny.md` will apply — but expect the **Evgeny A register** (long-form polished) to be the default for this channel, per the voice file's `<voice_fingerprint>` section.
